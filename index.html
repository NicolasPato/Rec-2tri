Os principais comandos para se criar um site

Com HTML5 e CSS

<head> - Define metadados como o título, estilos, scripts e outras informações que não são exibidas diretamente na página.

<body> - Define o corpo do site.

<p> - Paragrafo, define um bloco de texto que é exibido como um parágrafo.

<h1> - Título principal, geralmente usado para o título principal de uma página.

<h2> - Define o cabeçalho de nível 2, utilizado para seções importantes, mas subordinadas ao título principal.

<h3> - Define o cabeçalho de nível 3.

<h4> - Define o cabeçalho de nível 4.

<img> - É usada para incorporar uma imagem.

<a> - Define um hyperlink, que é usado para criar links para outras páginas.

<ol> - Define uma lista ordenada, onde os itens são numerados automaticamente.

<li> - Define um item de lista dentro de uma lista.

<title> - Define o título da página, que é exibido na aba do navegador.

color - Define a cor do texto em um elemento.

background-color - Define a cor de fundo de um elemento.

font-size - Define o tamanho da fonte do texto.

font-family - Especifica a família de fontes para o texto.

margin - Define a margem externa de um elemento, controlando o espaço ao redor dele.

padding - Define o preenchimento interno de um elemento, controlando o espaço dentro dele.

border - Define a borda de um elemento, especificando a largura, estilo e cor.

width - Define a largura de um elemento.

height - Define a altura de um elemento.

display - Especifica o tipo de caixa de renderização de um elemento, como block, inline, flex, etc.

position - Define o método de posicionamento de um elemento, como static, relative, absolute, fixed, ou sticky.
